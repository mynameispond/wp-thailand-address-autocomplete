Example: Add Address Sets via AJAX + jQuery CDN

How to use window.wpataInit(context)

  • window.wpataInit() or window.wpataInit(document) = init ทั้งหน้า
  • window.wpataInit('#sets-container') = ส่งเป็น selector แบบ id
  • window.wpataInit('.set') = ส่งเป็น selector แบบ class
  • window.wpataInit(document.getElementById('sets-container')) = ส่งเป็น DOM element
  • window.wpataInit($nodes) = ส่งเป็น jQuery object (แนะนำที่สุดหลัง append)
  • ไม่แนะนำให้ส่ง HTML string ตรงๆ เช่น window.wpataInit('<div>...')
  • window.wpataConfig.ajaxUrl มีค่า default อัตโนมัติอยู่แล้ว (ตั้งเองเฉพาะตอนอยาก override endpoint)
  • กรณี append ทีละชุดแล้ว init ทันที: ไม่ใส่ wpata-group-* ก็ได้
  • กรณี append หลายชุดพร้อมกันในครั้งเดียว: ควรใส่ wpata-group-*
